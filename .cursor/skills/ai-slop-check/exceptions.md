# Project exceptions (optional)

Edit after copying this skill folder into a product repo. Document fonts, colors, and patterns that **look** like slop but are **intentional** for your design system.

## Template

| Choice               | Intentional? | Notes                                                                        |
| -------------------- | ------------ | ---------------------------------------------------------------------------- |
| Primary font family  |              | e.g. one family OK if hierarchy is strong                                    |
| Dark theme           |              | Not slop if no neon glows                                                    |
| Card radius          |              | e.g. `rounded-lg` max on cards                                               |
| Accent color         |              | Brand accent ≠ purple AI palette                                             |
| shadcn/ui primitives |              | Default components fine; flag composition slop (nested cards, side-tab hack) |

## Still flag everywhere

- Purple/violet gradients as default palette
- Side-tab thick border on rounded cards
- Glassmorphism / blurred orbs as decoration
- Hero eyebrow + oversized long headline
- Nested cards
- Identical 3-col icon feature grids (unless marketing-only and on-brand)
- Bounce/spring UI motion
- Marketing buzzwords and em-dash spam
